Welcome to the Vehicle Selector!

Getting Started:
  1. Navigate to the Selector link at the top of the page.
  2. Search for and select a vehicle make.
  3. Search for and select a vehicle model.
Tech Stack:
Project's GitHub
  • Single Page Application (SPA)
    • Project Name: VehicleSelector.App
      • AngularJS/Bootstrap Frontend
        • Bower managed packages
      • Built on .Net Core 1.1
      • Hosted on Azure
  • Rest Application Programming Interface (API)
    • Project Name: VehicleSelector.Api
      • Built on .Net Core 1.1 (web api)
      • Hosted on Azure
    • Project Name: VehicleSelector.Data
      • Built on .Net Core 1.1 (class library)
      • Entity Framework Core 1.1.2
      • Hosted on Azure
  • SQL Database
    • Deployed via VehicleSelector.Data (code-first architecture)
    • Hosted on Azure
